# Diagnostic details

Technical details about the measurement, normalization choices, and caveats.

## Why normalize by $|L_{z,0}|$?

We normalize action variability by the initial angular momentum $|L_{z,0}|$ rather than by the initial action value itself. The stability metric is $\sigma(J) / |L_{z,0}|$, where $\sigma(J)$ is the temporal standard deviation of the action across all snapshots. This matters because:

- **$J_z$ starts small.** Particles are sampled near the midplane with realistic but small vertical excursions, so $J_{z,0}$ is small. A fractional measure $\sigma(J_z) / J_{z,0}$ would be dominated by numerical noise dividing by near-zero denominators.
- **$|L_z|$ is the natural action scale.** All actions have dimensions of angular momentum ($\mathrm{kpc} \cdot \mathrm{km/s}$). Dividing by $|L_z|$ makes both $\sigma(J_z)$ and $\sigma(J_R)$ dimensionless on the same footing.
- **Fair comparison.** The same normalization is applied to both actions, so the ratio $\sigma(J_z) / \sigma(J_R)$ is independent of the normalization choice.
- **Standard deviation vs. endpoint difference.** Using $\sigma(J)$ rather than $|J(t_\mathrm{end}) - J(t_0)|$ captures both secular drift and temporal fluctuations across all 50 snapshots, providing a more robust measure of how much each action varies during the simulation.

## Sentinel values and filtering

galpy's `actionAngleStaeckel` returns 9999.99 when the Staeckel fudge fails to converge. In our simulation, **zero convergence failures** occurred for either $J_z$ or $J_R$ across all 2000 particles and 50 snapshots. The action computation uses the median of the analytically-derived $\Delta(R)$ from the Miyamoto-Nagai separability condition (see [connection](../concepts/connection#deriving-deltar-from-the-separability-condition)), which provides a well-matched focal distance for the entire disk.

The plotting code still filters for sentinel values (any particle with $J_R \geq 9000$ is excluded) as a safety measure, but no particles are affected in this run.

## Scope and limitations

The simulation demonstrates algebraic protection of $J_z$ under **planar perturbations** -- specifically, the Dehnen bar acting within $SO(2)$. Two caveats should be noted:

1. **Gas and GMC scattering.** The potential is smooth and collisionless. In galaxies with gas, gravitational scattering off giant molecular clouds provides a non-planar perturbation channel that can drive vertical action evolution. [Arora et al. (2025)](https://doi.org/10.1093/mnras/stae2707) found that in MHD simulations with star formation, vertical actions evolve as a logarithmic random walk driven primarily by GMC encounters. This mechanism lies outside $SO(2)$ and is not protected by the Casimir argument.

2. **Bar strength and disc dominance.** [Mikkola, McMillan & Hobbs (2020)](https://doi.org/10.1093/mnras/staa364) showed that $J_z$ conservation degrades in disc-dominant simulations where the bar is very strong. In such regimes, the bar may excite vertical resonances that couple planar and vertical motion, partially breaking the separation assumed by the Staeckel fudge. Our MW-like potential, with a substantial halo, is in the regime where the algebraic protection holds.

## Reproducing the results

All figures on this site are generated by CLI commands that read from the simulation database:

```bash
# Download the simulation
uv run experiments download --environment development

# Or run a new one
uv run experiments simulate --name my-simulation --n-particles 2000

# Generate result figures
uv run experiments action-scatter --name mw # (or your own simulation)
uv run experiments migration-scatter --name mw
uv run experiments migration-plot --name mw

# Generate model figures
uv run experiments potential-plot
uv run experiments delta-plot
```
